# conversor-docx-html-papoca
Ferramenta criada em Python com Streamlit para converter arquivos .docx em código HTML, preservando a formatação original do documento. Criada para agilizar o processo de publicação de conteúdos no WordPress, reduzindo erros e economizando tempo no fluxo editorial.

## Tecnologias
- Python 3.12
- Streamlit
- Mammoth (conversão DOCX → HTML)

## Instalação das dependências
```bash
pip install -r requirements.txt
``` 

## Executar
```bash
streamlit run app.py
```
